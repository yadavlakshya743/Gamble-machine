# Python Slot Machine Game

A simple terminal-based slot machine game made with Python.

## How to Run

```
python slot_machine.py
```

## Features

- Bet on 1 to 3 lines
- Symbol-based payouts
- Manage balance
- Random slot results

## Symbols and Payouts

| Symbol | Count | Payout |
|--------|-------|--------|
| A      | 2     | 5x bet |
| B      | 4     | 4x bet |
| C      | 6     | 3x bet |
| D      | 8     | 2x bet |

## How to Play

1. Deposit money
2. Choose lines to bet on (1-3)
3. Set your bet per line
4. Spin and win if symbols match!

## Example

```
Deposit: $100
Lines: 2
Bet per line: $5

Slot:
C | D | B
A | A | A
D | C | B

You won $25 on lines: 2
```

## Requirements

- Python 3
